Instructions to run the code:
1. main1.py is the main program
2. train.txt and test.txt should be in the same directory as main1.py
3. output of data in test.txt will be saved in a text file name prediction.txt


Algorithm:

The algorithm now uses 2 as number of nearest neighbour. Change the value "num_neighbour" to vary the number of nearest neighbours. Pearson coefficient was used to measure the similarity between two users.
